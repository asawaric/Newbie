This Git_Help folder contains the tutorial on most important git commands useful to get started with git and work on your repositories (local and remote).

Each text file in this folder describes one topic ( e.g setting up a repository) which includes brief information of the purpose of the topic along with the links where reader can read details of the topic, checklist of the git commands and finally an exercise to test your complete understanding of the topic.

Some of the topics also include question on a given command under the command checklist section.
