How to delete a file or a folde from Remote repository?

1] git clone <SSH key of remote repository>

2] cd to <path/to/cloned directory>

3] git rm <foldername>; foldername == folder to be deleted

4] git commit -m <"message">

5] git rm <filename>; filename == file to be deleted

6] git commit -m <"message">

7] git pull -a <remote repository>

8] git push <remote repo> <branch>

For more details visit http://kaidez.com/remove-files-from-github/
